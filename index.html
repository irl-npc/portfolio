Prompt 01｜用户洞察与产品定位

用途
从商品资料与同类评价中提取痛点、场景、购买顾虑和可转化方向。

结构
角色 → 输入 → 任务 → 约束 → 输出

对应产出
用户痛点矩阵、产品定位与五条核心卖点。

你是一名电商内容策略师。请基于【商品原始资料】与【同类商品用户评价】完成分析：
1. 将评价信息区分为事实、主观感受与可推导需求；
2. 提炼 3—5 个高频用户痛点，并为每个痛点保留评价证据；
3. 将痛点映射为产品机会、使用场景和可表达卖点；
4. 单独列出包装、日期、价格等购买顾虑；
5. 不虚构成分、检测或功效，不使用“持久、彻底去除、绝不留痕”等无法验证的结论。

输出：①洞察摘要；②痛点/证据/机会三列表；③一句话定位；④五条稳妥卖点。

01 / 欢迎介绍直播间今天主要分享什么？
哈喽，欢迎刚刚进入直播间的朋友，我是今天的主播。今天主要给大家分享适合出租屋、宿舍和小户型使用的空间清新好物。如果你也会遇到房间闷、卫生间有异味，或者朋友突然来家里、担心一开门味道不太舒服，可以先在直播间停留一下。我们今天讲的是一间风空间清新喷雾，1号链接已经放好，一瓶 300ml，共有五种香型。
02 / 圈定人群哪些人会遇到这些气味问题？
直播间有没有住出租屋、宿舍或者小户型的朋友？这类房子里，卧室、卫生间、玄关和厨房离得比较近；做饭的味道、烟味、卫生间异味，还有不通风产生的闷味，很容易混在一起。尤其朋友突然说要来家里，房间虽然收拾干净了，还是会担心气味不够舒服。
03 / 产品卖点产品本身有哪些信息？
它帮助改善日常空间里的异味感受，让房间闻起来更清爽。第一，采用细雾喷头，喷出的水雾比较均匀；第二，有五种香型，可按卧室、卫生间、玄关和办公室等空间选择；第三，一瓶 300ml，打扫房间、朋友来访或回家前都可以使用。今天主推柑橘薰衣草：柑橘清爽，薰衣草更柔和，卧室、客厅和办公室都适合。
04 / 使用方法直播中怎样演示使用？
大家看一下喷头。使用时保持瓶身直立，朝房间内比较空旷的位置喷两到三下即可。不需要对着床、衣物和家具一直喷，也不要对着人、宠物或食物喷洒；使用后可以保持适当通风。小房间不必一次喷很多，可按空间大小和自己对香味浓淡的喜好调整用量。
05 / 场景描述不同空间怎样推荐？
比如下班回家，打开门觉得房间有一点闷，可以先开窗通风，再在玄关或房间喷几下。打扫完卫生间后，可选海洋薄荷这类清冽味道；卧室想要柔和一点，可选柑橘薰衣草或香草玉兰。重点不是把家里喷得特别香，而是走进不同空间时闻起来清清爽爽、比较舒服。

喜欢的朋友可以直接点开 1 号链接。日常单瓶价格 25.99 元，直播间单瓶到手 19.99 元；两瓶直播价格 29.99 元，香型任选。卧室选一瓶、卫生间选一瓶，可以按不同空间搭配。需要的朋友可在 1 号链接选择单瓶 19.99 元或两瓶 29.99 元规格。
07 / 香型组合不知道怎么选香型？
第一次购买、主要用于卧室客厅办公室，可选柑橘薰衣草；卫生间或玄关，选海洋薄荷；喜欢温暖居家感，选香草玉兰；喜欢清甜花香，选桃花玫瑰；书房和办公室推荐绿茶茉莉，气味更淡雅。两瓶装可选“柑橘薰衣草＋海洋薄荷”：一瓶用于卧室，一瓶用于卫生间。
08 / 强化优点如何重复强调核心卖点？
大家可以看一下喷头，按压后出来的是比较均匀的细雾，日常使用方便。一瓶 300ml，适合打扫完房间、朋友来访前，或回家觉得房间有些闷时使用。五种香型让不同空间不用全用一种味道：卧室柔和、卫生间清爽、书房淡雅。小户型先喷一到两下，再按房间大小调整。
09 / 场景推动怎样把组合带回生活场景？
两瓶装平均一瓶不到 15 元。一瓶柑橘薰衣草放在卧室，打扫完房间或朋友来之前喷几下；另一瓶海洋薄荷用于卫生间和玄关。不同香型做简单空间区分，适合生活区域离得近的出租屋、宿舍和小户型。
